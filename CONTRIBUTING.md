# 🤝 Contributing to Opening Forge

Thank you for considering contributing to Opening Forge.

Whether you're a chess player, developer, designer, or data enthusiast, contributions are welcome.

---

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/AyushSinha2603/opening-forge.git
cd opening-forge
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Project

```bash
python main.py
```

## Pull Request Guidelines

- Keep PRs focused on a single change.
- Use clear commit messages.
- Reference related issues.
- Add tests when possible.
- Update documentation if necessary.

---

## Development Workflow

1. Fork the repository
2. Create a branch

```bash
git checkout -b feature/my-feature
```

3. Make changes
4. Commit changes

```bash
git commit -m "Add feature"
```

5. Push your branch

```bash
git push origin feature/my-feature
```

6. Open a Pull Request
